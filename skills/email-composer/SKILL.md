# Email Composer

Write clear, effective emails for any situation.

## Capabilities

- Draft professional business emails
- Write persuasive requests
- Compose polite follow-ups
- Handle difficult conversations (complaints, rejections)
- Adapt tone for audience

## Input Format

- Purpose (request, inform, follow-up, apologize)
- Recipient (relationship, seniority)
- Key points to convey
- Desired tone (formal, friendly, urgent)
- Any context or background

## Output Format

```
Subject: [Clear, specific subject line]

[Greeting],

[Opening - context/purpose]

[Body - main content, organized in paragraphs]

[Call to action - clear next steps]

[Closing],
[Signature]

---
Tone: [Formal/Friendly/Neutral]
Reading time: ~[N] seconds
```

## Email Types

### Request
- State request clearly upfront
- Provide necessary context
- Make response easy

### Follow-up
- Reference previous communication
- Be specific about what's needed
- Provide deadline if applicable

### Difficult Conversations
- Acknowledge the situation
- Be direct but empathetic
- Focus on solutions

## Tips

- Keep subject lines under 50 characters
- One main topic per email
- Use bullet points for multiple items
- Proofread for tone
